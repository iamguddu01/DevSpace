# DevSpace

A full-stack web application featuring a modern React frontend and a robust Django/Python backend. Includes secure environment variable management, RESTful API integrations, and a clean user-interface.

## Project Structure
- `frontend/` - React frontend powered by Vite
- `backend/` - Django and Django REST Framework backend

## Setup Instructions

### Backend Setup
1. Navigate to the backend directory: `cd backend`
2. Create your environment variables file: `cp .env.example .env` and fill in your credentials.
3. Install dependencies and start the server as usual.

### Frontend Setup
1. Navigate to the frontend directory: `cd frontend`
2. Create your environment variables file if needed: `.env`
3. Run `npm install` and `npm run dev` to start the development server.
